## 📍 Sobre o Projeto

O **Flora UFR** é um projeto de extensão da **Universidade Federal de Rondonópolis (UFR)**, vinculado ao curso de **Engenharia Agrícola e Ambiental**. O campus apresenta um expressivo potencial paisagístico, porém sua gestão atual das áreas verdes é deficiente em aspectos técnicos e de planejamento.

Este projeto surge como resposta técnica e acadêmica para promover a revitalização sustentável, qualificando a manutenção, elaborando um Plano Diretor de Paisagismo e desenvolvendo ferramentas digitais para catalogação da flora.

### 🚨 Problemática Identificada
- Podas incorretas com abandono de tocos (risco de fungos e queda).
- Ausência de Plano Diretor de Paisagismo.
- Falta de catalogação digital das espécies.
- Degradação do gramado e déficit de sombreamento.

## 📱 Aplicativo Flora UFR

Este repositório contém o código fonte do protótipo do aplicativo web para catalogação das árvores do campus.

### Funcionalidades
- 🗺️ **Mapa Interativo:** Localização georreferenciada de cada árvore (Leaflet.js).
- 🌿 **Ficha Técnica:** Nome popular, científico, família e fotos.
- 📸 **Registro Colaborativo:** Upload de fotos de floração/frutificação.
- 🏷️ **QR Code:** Preparado para integração com plaquetas físicas.

### Tecnologias
- **Frontend:** HTML5, CSS3, JavaScript
- **Mapas:** Leaflet.js + OpenStreetMap
- **Dados:** CSV/JSON
- **Coordenadas Centrais:** `-16.465720, -54.578753`
